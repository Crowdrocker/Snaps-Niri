## üé® Custom Shortcuts

You can add your own shortcuts by editing:
```bash
~/.config/niri/keybinds.kdl
```

### Example: Add Custom Shortcut
```kdl
binds {
    Mod+Shift+B { spawn "brave"; }
}
```

## üí° Pro Tips

1. **Vim-style navigation**: Use `H J K L` instead of arrow keys
2. **Quick workspace switching**: Use number keys for instant access
3. **Gaming mode**: Press `Mod+G` before gaming for best performance
4. **Streaming mode**: Press `Mod+Shift+S` to optimize for streaming
5. **Center windows**: Use `Mod+C` to center the current column
6. **Floating windows**: Toggle with `Mod+Shift+Space` for dialogs
